# Prisme Bridge

Token Studio and Figma Variables speak different dialects. 
Prisme Bridge translates between them so your developers 
never have to rewrite a thing.

## What it does

Prisme Bridge is a bidirectional Figma plugin that syncs 
Token Studio JSON files with Figma Variables. Import your 
token files to create Figma variable collections, or export 
your Figma variables back to Token Studio JSON format.

## Features

- Import Token Studio JSON into Figma Variables
- Export Figma Variables back to Token Studio JSON
- Supports a three-tier token architecture: Global, Segment, and Semantic layers
- Handles COLOR, NUMBER, and STRING variable types
- Supports multiple modes per collection
- Preserves alias chains across collections

## How to use

### Import
1. Open the plugin in any Figma file
2. Select the collections you want to import
3. Choose your default segment
4. Click Import collections

### Export
1. Switch to the Export tab
2. Select the collections you want to export
3. Click Export collections
4. A zip file will download with one JSON file per collection
